# 🚀 Quick Start Guide

Get your portfolio live in 5 minutes!

## 📋 Prerequisites

- Modern web browser (Chrome, Firefox, Safari, Edge)
- Text editor (VS Code, Notepad++, Sublime Text)
- Your professional information ready

## ⚡ 5-Minute Setup

### Step 1: Open the Portfolio (1 min)
1. Navigate to the portfolio folder
2. Double-click `index.html`
3. Your portfolio opens in browser

**✅ Done!** Your portfolio is now live locally.

---

## 🎨 Customize in 5 Minutes

### Step 2: Edit Your Information (2 min)

**Using VS Code:**
1. Right-click on `index.html`
2. Select "Open with Code"
3. Find and replace:
   - `Brijesh Kumar Pandey` → Your Name
   - `Data Analyst` → Your Title
   - `brijesh.pandey@email.com` → Your Email
4. Save (Ctrl+S)
5. Refresh browser

**Using Notepad:**
1. Right-click on `index.html`
2. Select "Open with Notepad"
3. Find and replace text
4. Save
5. Refresh browser

### Step 3: Add Your Photo (2 min)

1. Find a professional photo (JPG or PNG)
2. Resize to 400x400 pixels
3. Save as `profile.jpg` in the `images/` folder
4. Refresh browser

**💡 Quick tip:** Use [Canva](https://canva.com) to crop/resize images instantly

## 🌐 Deploy (Choose One)

### Option 1: GitHub Pages (Free, Recommended)

```bash
# 1. Create account at github.com
# 2. Create new repository named: yourusername.github.io
# 3. Clone repository
git clone https://github.com/yourusername/yourusername.github.io
# 4. Copy portfolio files to the folder
# 5. Push to GitHub
git add .
git commit -m "Add portfolio"
git push origin main
# 6. Visit: yourusername.github.io
```

### Option 2: Netlify (Free, Easiest)

1. Go to [netlify.com](https://netlify.com)
2. Sign up with GitHub
3. Click "New site from Git"
4. Select your repository
5. Click "Deploy"
6. Get your free domain

### Option 3: Vercel (Free)

1. Go to [vercel.com](https://vercel.com)
2. Sign up with GitHub
3. Click "Import Project"
4. Select your repository
5. Deploy automatically

### Option 4: Traditional Web Server (Paid)

1. Upload all files via FTP
2. Access via your domain
3. Done!

## ✨ Key Customizations

### Change Colors

In `styles.css`, find and change:
```css
from-cyan-400   →  from-purple-500   (for purple theme)
to-blue-500     →  to-pink-500       (for pink theme)
```

### Update Social Links

Find these lines and add your links:
```html
<a href="#"><i class="fab fa-linkedin"></i></a>
```

Replace `#` with:
- LinkedIn: `https://linkedin.com/in/yourprofile`
- GitHub: `https://github.com/yourprofile`
- Twitter: `https://twitter.com/yourhandle`

### Add More Experience

Copy this block in the Experience section:
```html
<div class="experience-card">
    <h3>Your Job Title</h3>
    <p>Company Name, City</p>
    <span>Jan 2020 - Present</span>
    <ul>
        <li>Achievement 1</li>
        <li>Achievement 2</li>
    </ul>
</div>
```
