# what is operator in python ?

  **def**
  1. operator operand some action there we applied operator
   
     examples : addition | substractions | compare | conditional value check

  **types of operator**

  1. airthematic operator 

     performed some airthematical expression there we used airthematic operator     

     examples : + , -, *, / , % etc
  
  2. comparision operator 

     where we compare with 2 values either its true or false applied comparision operator      

     examples : >, >=, <, <=, ===  etc
     
  3. assingments  operator 

     where we compare and assign two values and check condition there we applied assignment operator      

     examples : =, ==, !=  etc
  
   4. logical operator 

     where check and compare two values if both are true applied and if one true and one false applied or 

     examples : and | or 

   5. ternary or conditional operator 

      python is not official used ternary operator but yes we used conditional operator here 

      examples : if true "" else "false"

   6. betwise operator :

      examples : +=, -= , *= , %= , /=, >> , <<
   
 
   7. increment/decrement operator

      examples :i=i+1
                i=i-1

     **note: python not support pre increment/decrement**
