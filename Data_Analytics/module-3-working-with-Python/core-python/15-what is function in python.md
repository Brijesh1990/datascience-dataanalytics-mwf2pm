# what is function in python ?

  1. A function is a block of coded that can be used o completed any task there we used function
  2. A function is re-usables one block to another block 
  3. A function is created in python using **def keyword**
  4. A function is block of code just like 

     ```
      def functionname():
        statements
      call a function

     ```

# arguments or parameter in function 

  1. arguments or parameter in function pass single arguments
  
  ```
  def name(fname):
    statements
  call a function 
  ```
  2. arguments or parameter in function pass multiple arguments

  ```
  def name(fnm,lnm):
    statements
  call a function 
  ```
