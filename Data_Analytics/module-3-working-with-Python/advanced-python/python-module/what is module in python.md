# what is module in python ? 

  1. A python module is save with .py or a small peace of .py file i.e called module in python
  2. A module is reusable
  3. A module is re-used one to another file 
  4. A module is some types i.e inbuilt module or user defined or third party module 
  5. A module is always used with **import** keyword 
  6. A module is used light weight programmes 
  7. A module is downloadable if we used third party using **pip**
  8. A pip is python install package used to install any third party module

# strftime all parameter 

| Directive | Meaning | Output Example |
|-----------|---------|----------------|
| `%Y` | Year with century, as a decimal number | `2026` |
| `%y` | Year without century, as a decimal number (00–99) | `26` |
| `%m` | Month as a zero-padded decimal number | `07` |
| `%B` | Locale’s full month name | `July` |
| `%b` | Locale’s abbreviated month name | `Jul` |
