# what is file handling in python ? 

  1. file handling is used to handel a file 
  2. file handling is used to open | close | read | write on file there we used file handling 
  3. file handling is used some modes to handel any file in python. 
  4. file handling used some function to handel any file in python.


## file handling function     

   - open()
   - close()
   - write()
   - read()
   - with()


## file handling mode 

  - r: read only 
  - r+ : read | write only 
  - w: write only but if file does not exist create an new file 
  - w+: read | write but if file does not available create new file 
  - a : append on file if file does not available create new file 
  - a+: append and read  file if file does not available create new file 
 
## how to check mode 

  - check mode of file using mode method
